**lambda表达式其实就是用来简洁表达一行就完的函数的表达式**

举例：

```python
# def f(x):
#	 return x*x
# 那么这个函数就可以写作lambda表达式
f = lambda x:x*x	# 

print(f(2))
```

- `:` 之前的参数是lambda表达式的输入
- `:` 之后的式子是lambda表达式的输出
